# 📌 Project Overview

The **Australian Retail Analysis Dashboard** is an interactive Business Intelligence project built using **Microsoft Power BI** to analyze retail sales performance across Australia.

The dashboard processes **80,962 retail transactions** from **January 2016 to August 2017**, covering **2 retail chains**, **10 product categories**, **21 store managers**, **98 store locations**, and **8 Australian states**.

This project demonstrates advanced Power BI concepts including:

- ⭐ Star Schema Data Model
- 🔄 Power Query Append Queries
- 📊 DAX Measures
- 🤖 Power BI Q&A AI Visual
- 📅 Financial Year Intelligence
- 📈 Interactive Dashboard Design

---
